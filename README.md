## 💻 Mi flujo de trabajo
Al trabajar entre los equipos del instituto y mi PC personal, mi flujo de trabajo habitual con Git para mantener los repositorios actualizados y sincronizados es el siguiente:

```bash
# Sincronizo los últimos cambios antes de empezar a trabajar
git pull origin main

# Añado las nuevas prácticas y scripts realizados
git add .
git commit -m "Actualización de prácticas"

# Subo los cambios al repositorio central
git push origin main
```
